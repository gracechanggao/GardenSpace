Angular 2 component for the WjFlexChart ChartAnimation object.

The wj-flex-chart-animation component must be contained in a WjFlexChart or WjFlexPie component.

Use the wj-flex-chart-animation component to add ChartAnimation objects to your Angular 2 applications. For details about Angular 2 markup syntax, see Angular 2 Markup.

The WjFlexChartAnimation component is derived from the ChartAnimation class and inherits all its properties, events and methods.
